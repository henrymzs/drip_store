## Como clonar o repositório

Para clonar o repositório, use o seguinte comando no terminal:

```bash
git clone https://github.com/usuario/nome-do-repositorio.git
```

Instalar Dependências
```bash
npm install
```

Inicializando o projeto
```bash
npm run dev
```
